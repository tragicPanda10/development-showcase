Animal Crossing Fish Aggregator

Animal Crossing is a massively popular social simulation game where players can run around, build houses, farm, collect items, talk with locals, become part of a town, and more.

One aspect of the game is fishing. There are 80 different fish to catch in the game, so trying to catch them all is no easy feat. Using APIs and React, we can make this process a little easier to navigate.

In-game image of a fish collection

Objective

Create a fish aggregator which will help players find specific fish to go catch in the game.

ACNH API

Animal Crossing New Horizons (ACNH) API is a publicly available API that contains data for nearly all the assets found in the game. I used this API to get all the available information about all the fish for the aggregator.

I decided to only include the price, rarity, location, and description of each fish because I considered these to be the most important details for someone looking to make a list of fish to catch.

Filters and Sorting

The aggregator can filter by rarity or location or both, making it easy to find fish based on the in-game area you're in.

By default, the fish are sorted alphabetically. However, you can sort by price or by increasing rarity.
